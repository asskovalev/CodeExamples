Autostub
========

Интерцептор unity для автоматической генерации xml стабов для методов интерфейса с инъекцией и глушения при последующих вызовах


ActiveCache
========

прототипирование шедулера запуска шагов бизнес-процесса на основе очереди с приоритетами


CallParser
========

Утилита разбора логов в читаемое дерево вызовов (хардкод под конкретный проект)


gsmParser
========

тест парсера gsm/cdma файлов


RollingAverage
========

прототипирование скользящего и накапливающего среднего для использования в модуле сбора статистики производительности
